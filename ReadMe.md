# Update Note Testing content repository

This repo is being used to test and trial creating update notes using markdown. Please check with the [repo owners](#repository-owners) for current state of trials.

---

## Reading the content

You can read the content in this repository in:

- GitHub starting with the [Home page](https://github.com/WiseTechGlobal/UPNTest.Content/blob/main/Home/Update-note-test-repo.md).
- At some point in the future, in [WiseTech Academy](https://wisetechacademy.com/) as Update Notes.

There are no current plans to publish this to SharePoint.
